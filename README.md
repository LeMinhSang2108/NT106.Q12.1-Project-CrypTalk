## Yêu cầu hệ thống / Prerequisites

Trước khi cài đặt, hãy đảm bảo máy tính của bạn đã đáp ứng yêu cầu sau:
> **Core Requirement:** .NET 8.0 (SDK or Runtime) trở lên.

* Kiểm tra bằng lệnh: `dotnet --version`
* Nếu chưa có, tải tại: [Download .NET 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

## Cài đặt / Installation

Bạn có thể chạy ứng dụng bằng cách tải bản đóng gói (Recommended) hoặc chạy từ mã nguồn.

### Tiếng Việt

#### Cách 1: Dành cho người bình thường muốn cài và sử dụng ngay (Khuyên dùng)
Phương pháp này đơn giản nhất, không cần cài đặt phức tạp.

1.  Truy cập mục **[Releases](../../releases)** của repository này.
2.  Tải xuống file cài đặt `CrypTalk_Setup_Final_v1.0.exe` mới nhất.
3.  Chạy file trên, cài đặt như 1 ứng dụng bình thường (điều mà bản thân người viết nghĩ ai cũng có thể làm được trong năm 2026).
4.  Đăng nhập và bắt đầu trải nghiệm ứng dụng.

#### Cách 2: Dành cho người muốn hiểu tường tận kiến trúc hệ thống của ứng dụng này (Chạy từ Source Code)
1.  Clone dự án về máy:
    ```bash
    git clone (https://github.com/USERNAME/Cryptalk_NT106.git)
    ```
2.  Mở Solution bằng **Visual Studio 2022**.
3.  Nhấn `F5` để Build và Run project.

### English

#### Option 1: For Normal Users (Recommended)
The easiest way to run CrypTalk without complex setup.

1.  Go to the **[Releases](../../releases)** section of this repository.
2.  Download `CrypTalk_Setup_Final_v1.0.exe` and install the latest version.
3.  Just run it and install like a normal software.
4.  Login and start chatting!

#### Option 2: For Dev or Reverse Engineer enjoyer (Run from Source)
1.  Clone the repository:
    ```bash
    git clone (https://github.com/USERNAME/Cryptalk_NT106.git)
    ```
2.  Open the Solution file in **Visual Studio 2022**.
3.  Press `F5` to Build and Run.
